# 💻Electronics Store Database

This repository contains the database schema for an electronics retail store. It includes tables for managing products, categories, suppliers, stores, employees, customers, and discount cards.

⭐Database Structure
The database consists of the following tables:
· Categories – product categories
· Providers – suppliers
· Tovar (Products) – product catalog
· tovarsShops – many-to-many relationship between products and stores (stock availability)
· tovarsBuyers – many-to-many relationship between products and buyers (purchase history)
· Shops – store locations
· Employees – store employees
· Position – employee's work position
· Buyers – customers
· discondCards – discount cards
